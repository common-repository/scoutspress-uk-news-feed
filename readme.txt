UK Scout Association news feed widget & shortcode.

== Description ==
This plugin provides both a widget and shortcode to display posts from the UK Scout Association news feed on your website.  It has options to limit the number of posts shown and can be used either in conjunction with our ScoutsPress.com template or as a standalone feature.

== Installation ==
1. Download the plugin from WordPress.org or ScoutsPress.com.
2. Unzip the package and move to your plugins directory.
3. Log into WordPress and navigate to the "Plugins" screen.
4. Locate "Scouts News Feed" in the list and click the "Activate" link.

Done
